Mobile ready navigation

Add class .app-navigation-moblie-wrapper to .app to get this feature working.

Then you can use navigation-mobile markup inside .app to add your navigation.

Add data-navigation-mobile-toggle="true" to any button to open navigation.

Important! This is simple navigation, so it's not supports any other features. But it can be customized by yourself.
